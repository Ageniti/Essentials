# Code of Conduct

## Our Standard

This project aims to keep collaboration respectful, direct, and constructive.

Examples of expected behavior:

- being respectful in technical disagreement
- giving clear, actionable review feedback
- assuming good intent while staying precise
- focusing criticism on code and design, not people

Examples of unacceptable behavior:

- harassment or personal attacks
- discriminatory language or behavior
- repeated hostility in issues, reviews, or discussions
- intentionally disruptive or bad-faith participation

## Enforcement

Project maintainers may remove, edit, or reject contributions, comments, or
interactions that violate these standards.

## Scope

This code of conduct applies in project spaces, including issues, pull
requests, review comments, and repository discussions.
