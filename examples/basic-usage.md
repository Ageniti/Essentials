# Basic Usage

## Start the interactive runtime

```bash
essentials
```

## Run a single prompt

```bash
essentials -p "summarize this repository"
```

## Use a settings file

```bash
essentials --settings ./settings.json
```

## Dry-run a prompt

```bash
essentials --dry-run -p "inspect the available tools"
```

## Resume a session

```bash
essentials resume
```
